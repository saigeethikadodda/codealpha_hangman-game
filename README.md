🎮 Hangman Game

A simple text-based Hangman Game built using Python. The player tries to guess a hidden word one letter at a time before running out of chances.

📌 Project Description

This project is a console-based implementation of the classic Hangman game. A random word is selected from a predefined list, and the player must guess the word by entering letters. The player has a maximum of 6 incorrect guesses before the game ends.

✨ Features
Random word selection from a predefined list
Interactive console-based gameplay
Tracks guessed letters
Input validation
Displays Hangman stages
Win and Lose conditions
Beginner-friendly Python project

🎯 Game Rules
A random word is selected.
The player guesses one letter at a time.
Correct guesses reveal the letter's position.
Incorrect guesses reduce remaining chances.
The player has a maximum of 6 wrong guesses.
Guess the word before the chances run out to win.
